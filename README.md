# Vision Transformer Image Classification

## Descrição

Este projeto apresenta uma comparação entre duas arquiteturas Vision Transformer (ViT):

- ViT Base
- ViT Large

O objetivo é comparar:

- classe prevista;
- probabilidade da classificação;
- tempo de inferência.

Todos os experimentos foram realizados utilizando Google Colab, Python e a biblioteca Hugging Face Transformers.
